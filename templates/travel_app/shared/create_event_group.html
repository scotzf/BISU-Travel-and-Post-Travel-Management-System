What is an Event Group?

An Event Group links multiple travel records that are for the same event but were created separately by different employees.

For example, if 3 employees from different colleges all traveled to the same conference but created their own travel records, you can group them here so the system knows they're part of the same trip.

Grouping prevents double-counting in statistics and shows the true total cost of an event.
